## 환경변수

```bash
SERVER_PORT=서버포트
DATABASE_HOST=DB호스트
DATABASE_PORT=DB포트
DATABASE_USERNAME=DB사용자이름
DATABASE_PASSWORD=DB비밀번호
DATABASE_NAME=DB이름
NAVER_ID=네이버아이디
NAVER_PW=네이버 비밀번호
```

## 실행

```bash
$ npm install

# development mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
